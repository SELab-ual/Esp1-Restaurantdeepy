# RMOS - Restaurant Menu & Ordering System
## Sprint 1 Prototype: Digital Menu Foundation

### Overview
This prototype implements the core foundation for a digital restaurant menu system. Customers can view menu items by category, build orders, and place them. Orders are stored in the database for waiter access.

### Implemented Features
✅ Digital menu display with categories
✅ Add/remove items to order
✅ Quantity adjustment
✅ Order total calculation
✅ Order submission to backend
✅ Table number selection
✅ Call waiter functionality (UI only)
✅ Responsive design for tablets/phones

### Technology Stack
- **Backend**: FastAPI (Python 3.11)
- **Database**: PostgreSQL 15
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Proxy**: Nginx
- **Container**: Docker & Docker Compose

### Prerequisites
- Docker 20.10+
- Docker Compose 2.0+
- 4GB+ RAM available
- Ports 80, 8000, 5432 available
